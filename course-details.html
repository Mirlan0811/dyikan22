Курстун чоо-жайы

Сабак – окуучулардын үйрөнүлүп жаткан материалды (билим, жөндөмдүүлүк, билгичтик, идеялык жана моралдык-эстетикалык идеяларды) өздөштүрүү максаты менен окутууну уюштуруунун формасы. Бул форма класстык-сабактык окутуу системасында колдонулат жана класс үчүн, башкача айтканда, салыштырмалуу туруктуу билим берүү жамааты үчүн жүргүзүлөт

Что такое ООП в JavaScript

Негизи, OOP - бул бир объектти колдонуу менен объекттерди түзүүгө мүмкүндүк берген код жазуу ыкмасы. Бул Конструктор дизайн үлгүсүнүн да маңызы. Бир топ убакыт өткөндөн кийин, объектке багытталган программалоо (OOP) пайда болду, ал функцияларды жана маалыматтарды бир объектке - "объектке" топтоштурууга мүмкүндүк берет. Объектке багытталган иштеп чыгууда биз түзүлүп жаткан объекттердин деңгээлинде эмне болуп жатканын, алардын касиеттерин өзгөртүп, бири-бири менен жана (браузердин учурда) барак менен, жалпысынан, түз байланышта болгонун сүрөттөйбүз. Мисалы, “колдонуучу”, “меню”, “интерфейс компоненти”... Объектке багытталган мамиледе ар бир объект методдору жана маалыматтары бар интуитивдик объект болушу керек

Өзгөрмөлөр - маалыматтарды сактоо үчүн контейнерлер болуп саналат

JavaScript өзгөрмөлөрү төрт жол менен жарыяланышы мүмкүн:
Автоматтык түрдө
var колдонуу
let колдонуу
const колдонуу

var ачкыч сөзү 1995-жылдан 2015-жылга чейин бардык JavaScript коддорунда колдонулган.
let жана const ачкыч сөздөрү 2015-жылы JavaScript'ке кошулган.
var ачкыч сөзү эски браузерлер үчүн жазылган коддо гана колдонулушу керек.

JavaScript операторлору

Javascript операторлору ар кандай математикалык жана логикалык эсептөөлөрдү аткаруу үчүн колдонулат

Дайындоо оператору = маанилерди дайындайт
Кошуу оператору + маанилерди кошот
Көбөйтүү оператору * маанилерди көбөйтөт
Салыштыруу оператору > маанилерди салыштырат
